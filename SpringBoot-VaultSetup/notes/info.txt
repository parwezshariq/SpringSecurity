## Setup Hashicorp Vault

1) Download from https://www.vaultproject.io/ > Unzip

2) Add vault.conf file in unzipped folder

** vault.conf **

*************************************************

backend "inmem" {

}

listener "tcp" {
	address = "0.0.0.0:8200"
	tls_disable = 1
}

disable_mlock = true

*************************************************

3) Run following cmd commands:

C:\DevTools\vault_1.4.3_windows_amd64> vault server -config vault.conf

This will start vault server

** Open another terminal

C:\DevTools\vault_1.4.3_windows_amd64>set VAULT_ADDR=http://localhost:8200
